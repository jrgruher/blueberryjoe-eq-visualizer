🫐 BlueberryEQ — How It Works v0.2

A self-contained internet-radio player with a classic stereo-amp spectrum analyzer. It streams the station itself and reads the audio's frequency data directly — no plugins, no loopback cable, no install. Everything lives in this one HTML file.

Getting started

  • Click a station in the left list, then press ▶ Play (browsers need one click before audio can start).
  • Use Space to play/pause, ←/→ to change station, F for fullscreen.
  • Double-click the visualizer to go fullscreen; controls and cursor auto-hide.

Hotkeys

  • Space play/pause · ←/→ change station · F fullscreen
  • ↑/↓ volume · +/- gain
  • [/] bars · {/} rows · ,/. brightness
  • t/T cycle theme · c color mode · r reset view
  • All hotkeys work in fullscreen — a small label shows the change.

Stations

  • Genre filter (All / D&B / Lofi / Hip-Hop) narrows the list. Added streams are auto-sorted by name.
  • Add a stream: paste a stream URL in the bottom box and click ＋. Paste a .pls/.m3u URL, drag a playlist file onto the window, or use 📁.
  • hide offline drops stations that failed to load. A red dot + ⚠ marks a dead/unavailable stream — click it again to retry (these are community streams that go up and down).
  • ↺ Reset restores the built-in station list.
  • ⬆ / ⬇ export or import just your station list as a JSON file (import can replace or merge).

Visualizer controls

  • Vol — audio output volume (separate from Gain).
  • Gain — visual sensitivity. Higher = taller bars.
  • Bars — number of frequency columns (higher = finer resolution).
  • Rows — LED block density per bar.
  • Bright — color brightness, to match your display.
  • Theme — color palette (Classic, Fire, Ice, Rainbow, etc.).
  • Color by — Level colors each bar by height; Bars colors across the frequency spectrum.

Saving & sharing

  • All settings and your station list auto-save in this browser — they're restored next time you open the file.
  • 💾 Save downloads eq-settings.json (controls and stations) as a backup or to share.
  • 📂 Load restores from that file. To share with a friend, send them this HTML file (and optionally your eq-settings.json).

Good to know

  • Some stations stream over http://. They work when you open this file directly (the normal way). If you ever host the page on https://, browsers block those — the https stations still work.
  • A station only drives the bars if its server allows cross-origin reads (all built-in stations are pre-verified).
